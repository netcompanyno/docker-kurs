Oppsett Docker deamon+cli
======

Får å kunne kjøre containere på lokal klient må man installere en Docker deamon.
I disse oppgavene skal vi i tillegg bruke Docker CLI til å bygge, kjøre og gjøre opperasjoner mot containere.

#### Instruksjoner for installasjon

[Windows](https://docs.docker.com/docker-for-windows/install/) - Docker Desktop

- Last ned docker desktop installasjonsfil fra Docker Hub
- Sørg for å aktivere BIOS-level hardware virtualization
 

[Mac](https://docs.docker.com/docker-for-mac/install/) - Docker Desktop

- Last ned Docker Desktop installasjonsfil fra Docker Hub
- Drag and drop? :)

[Linux (Ubuntu)](https://docs.docker.com/engine/install/ubuntu/) - Docker Engine

- Fjern gamle versjoner
- Legg til repository
